The Chairman Bridge

The central orchestration hub. It manages the primary polling loops for chat sessions and coordinates the lifecycle of all specialized sub-modules.

chairman-bridge.ts LLM Orchestration

Terminal Monitor

Watches PTY output streams for high-stakes permission prompts (like Claude Code's "Allow run command?"). It intercepts these and posts interactive approval cards to chat.

terminal-monitor.ts Screen Scraping

Message Bridge

Ensures agents never miss critical @mentions. If a message is posted in chat but the agent is "lost" in their terminal, the bridge injects the message directly into the terminal UI.

message-bridge.ts Injection Logic

Task Watchdog

The system's integrity layer. It detects idle agents, identifies "silent failures," and ensures that tasks marked as complete actually align with terminal exit states.
