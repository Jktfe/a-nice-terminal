# Pi Local Adapter

Read `AGENTS.md` first. It is the canonical ANT vNext project contract.

Pi is a local runtime lane. Superpowers for Pi should come from the pinned
manifest at `superpowers/sync-manifest.json` and the local mirror at
`.ant-runtime/superpowers/current`, not from a live branch fetch during a room
run.

Run:

```sh
bun run superpowers:drift
```

before claiming Pi terminal skills or tools are current.
