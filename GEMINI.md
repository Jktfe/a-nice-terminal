# Gemini / Antigravity Adapter

Read `AGENTS.md` first. It is the canonical ANT vNext project contract.

Superpowers are pinned in `superpowers/sync-manifest.json`. The upstream
Superpowers repo includes a `GEMINI.md`; ANT mirrors that file under
`.ant-runtime/superpowers/current/adapters/GEMINI.md` when sync is run with
`--write`.

Keep Gemini or Antigravity-specific setup here. Shared repo rules stay in
`AGENTS.md`.
