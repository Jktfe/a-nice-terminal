# ANT Crawler Asset Pack

Premium SVG crawler assets derived from the supplied ant artwork at
`/Users/jamesking/Downloads/ant.svg`.

- `ant-crawler-rig.svg` is the assembled animation rig.
- `demo.html` is the visual demo page. It loads the rig and toggles crawl/still states.
- `ant-crawler-elements.svg` is the element contact sheet.
- `elements/*.svg` are standalone SVG exports for individual body and limb parts.

The rig is static by default. Add `is-crawling` to the root SVG class to run the
built-in crawl cycle.
