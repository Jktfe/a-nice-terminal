Non-negotiables

7 rules · override everything below
  • The Raw Terminal is byte-faithful. No structured layer can drop, reorder, or summarise PTY bytes.
  • Timestamps are required signal on every event.
  • trust:'raw' bytes never render as rich content.
  • Status displays separately from the timeline.
  • The user's intent is the main grid.
  • Trust belongs to events, not agents.
  • No code is shipped from ANTstorm.

Source-of-truth architecture

one log, four projections

Canonical truth = append-only JSONL log per session on disk. The only artifact that cannot be lost or rebuilt from elsewhere.

SQL projections (run_events, command_events, FTS5) rebuilt from the log via a single writeLogAndProject() function.

Markdown views are derived — generated from the log on demand, never the source of truth.

Artifacts content-addressed by SHA-256, served via /api/artifacts/:hash, referenced from run_events.payload.

Track 2 — RAW Browser Terminal

the contract that lets Track 1 be beautiful without becoming misleading
  • axterm.js is the live-tail renderer, not the transcript database. Bounded scrollback only.
  • bCanonical log + SQLite own transcript/search. Deep history is virtualized, never replayed into xterm.
  • ctmux is recovery/control, not source of truth. Reconnect events tagged source:'tmux' trust:'raw' with byte-range gap recorded.
  • dFive explicit scroll-anchor states: following, user_scrolled, replaying, alt_screen, hidden.
  • eBackpressure is explicit: terminal_throttle → tmux %pause / %continue. Dropping bytes is never valid.
  • fDOM renderer default; WebGL behind feature flag, after document.fonts.ready, fallback on contextlost.
  • gtmux allow-passthrough OFF by default. Phase 2 only with regression tests proving stability.
  • hNine failure modes named for direct test: duplicate replay, missing bytes during throttle, scroll jumps mid-read, alt-screen corruption, font/WebGL race, hidden-tab burst, clear-screen flicker, capture truncation, mobile resize.
