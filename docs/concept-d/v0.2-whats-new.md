# What's new in remoteant v0.2

> A full redesign of the Mac client around **Concept D** — your work day in one window: sources on the left, what needs you today in the middle, the room you're actually in on the right. Light mode, native Mac chrome, keyboard-first.

---

## The Workspace

Three columns, all collapsible, all remembered:

- **Sources** — Today · Asks · Rooms · Library · Agents · Vault · Memory · plus your saved rooms (drag-to-reorder, ★ pin) and a quick line to Finder · Calendar · Shortcuts on this Mac
- **Today** — the asks needing you · the warm rooms · the plans in progress · all live, all in glanceable depth
- **Room** — the chat you're working in, with the room shelf on the right tracking artefacts · plan steps · interviews · memories · attachments · linked rooms · (and ★ Chair + ★ Validation as premium tabs)

Every panel folds down with one click and comes back from the toolbar. No dead-end collapses.

---

## Bring in another assistant

The room header now has one-tap chips for **Claude Desktop** · **Claude Mobile** · **ChatGPT** · **Gemini**. Click one, the room's context lands in that app, you work there, results come back to ANT.

> Working on the train, you tap "Bring in ChatGPT" on your iPhone, ask it to draft three social posts using the brief from this room, review them, approve the one you like — it posts to the room.

(The full round-trip with auto-return + Approve/Reject queue lands in v0.3. v0.2 ships the deep-links.)

---

## Native to the Mac

This isn't a Chrome tab pretending to be a desktop app.

- **NSToolbar** with the `>_ANT` brand mark, ⌘K search palette, notifications, share, profile
- **Drag any file from Finder** — PDFs, Numbers, Pages, Keynote, screenshots — into a room. Evidence-aware (drop-target wiring full in v0.3; drop zone visible in v0.2)
- **Native bridges strip** at the bottom of the window: Mail · Calendar · Reminders · Notes · Safari · Chrome · Teams · Zoom · Office · iWork · Files (chips in v0.2; full drag-and-drop in v0.3)
- **Calendar plan-step sync** and **Reminders ask-escalation** are wired conceptually — see v0.3 for permissions handshake
- **Continuity Camera, Dictate, Quick Look, Share Sheet** — all native macOS, no in-app reinvention
- **Window restore**, **column-width persistence**, **keyboard shortcuts in the menu bar** — Mail.app-grade polish

## Keyboard

Built for keyboard-first operators.

| Shortcut | Action |
|---|---|
| ⌘1 / ⌘B | Toggle sidebar |
| ⌘2 | Toggle Today column |
| ⌘3 | Toggle room shelf |
| ⌘⇧B | Toggle native bridges strip |
| ⌘K | Focus search palette |
| ⌘⇧4 | Screenshot the active room |
| ⌘↩ | Send composer message |
| `/break <label>` | Post a break in the room |

## Idiot-proof

A non-engineer on your team should be able to use this on day one. Five sources in the sidebar with plain English labels. One big **"Start something"** button at the bottom of the sidebar — that's how you create a new room. ★ premium tabs are visible-but-locked so paid features don't hide behind menu trees. VoiceOver labels on every interactive element. Pointer-drag reorder for saved rooms.

---

## Premium (★)

- **★ Chair** — long-running session tracker. Knows when an agent has been working for hours and gives you a digest instead of the raw replay.
- **★ Validation** — claim extraction + automated %-score against your verification policies. The structural fix for "agent extrapolated one example to a universal claim."

Both render as locked tabs in v0.2 with a clear upgrade path. v0.3 wires the subscription gate via accounts.antonline.dev.

## Install

```sh
# fresh install on macOS
brew install jktfe/antchat/ant

# upgrading from an earlier remoteant
brew upgrade jktfe/antchat/ant
```

Or grab the signed DMG directly from [antonline.dev/downloads](https://antonline.dev/downloads).

First run signs you in. Two paths:
- **Team Login** — email + password + licence key (if you have an `accounts.antonline.dev` subscription)
- **Invite Token** — server URL + room ID + token (if a teammate sent you one)

Existing v0.1.x users will be migrated transparently. Server-side state unchanged.

## What's deferred to v0.3

Honest about what's not here yet:

- Functional drag-drop into native bridge chips (chips render, drops are no-ops)
- Bring-in-LLM auto-return + Review queue (deep-links work, return-trip is manual via copy-paste for v0.2)
- Multi-line composer + paperclip + Continuity Camera button in composer + Dictate button + per-message router picker
- Break-delete from the chat stream (needs a server endpoint that ships first)
- Premium feature wiring (★ Chair tracker logic, ★ Validation scoring engine) — visible-locked in v0.2
- VoiceOver custom-action reorder on saved-rooms list (pointer drag works today)

If any of these matter to you for v0.2: drop a note in [issues on Jktfe/a-nice-terminal](https://github.com/Jktfe/a-nice-terminal/issues) — we'll prioritise.
